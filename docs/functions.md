# 機能一覧

## Frontend

| 大分類       | コメント                                         |
| ------------ | ------------------------------------------------ |
| ログイン機能 | メール＋パスワードログイン、ログアウト           |
| チャットUI   | ユーザー発言／AIの返答                           |
| 会話履歴管理 | 会話履歴の保存・表示・削除、セッションの切り替え |
| システム設定 | モデル選択                                       |

## Backend

| 大分類       | コメント                   |
| ------------ | -------------------------- |
| 認証機能     | Cognito のユーザ認証       |
| ユーザ管理   | ユーザ情報の登録・更新     |
| 会話履歴管理 | 会話履歴の保存・表示・削除 |
